# StratifySDK
CMake Project to Pull and build the Stratify SDK (Stratify OS, Stratify API and supporting libraries)

On Mac:

```
git clone https://github.com/StratifyLabs/StratifySDK.git
cd StratifySDK
mkdir build
cd build
cmake ..
```

On Windows:

```
git clone https://github.com/StratifyLabs/StratifySDK.git
cd StratifySDK
mkdir build
cd build
cmake .. -G "MinGW Makefiles"
```
